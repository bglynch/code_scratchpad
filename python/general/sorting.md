# Sorting

Sorting a list

```python
li = [9,1,8,2,7,3,6,4,5]

sorted(li)
>>> [1, 2, 3, 4, 5, 6, 7, 8, 9]

sorted(li, reverse=True)
>>> [9, 8, 7, 6, 5, 4, 3, 2, 1]


```

Sorting a dict returns a list of sorted keys

```python
di = {'name': 'Brian', 'job':'software', 'age': None, 'os':'Mac'}

sorted(di)
>>> ['age', 'job', 'name', 'os']
```
